TROUBLESHOOTING:
===========================================

Q: MySQL service won't start?
A: 1. Open Services (Windows+R ‚Üí services.msc)
   2. Find MySQL80
   3. Right-click ‚Üí Properties
   4. Click "Start" button
   5. If error, try "Restart"

Q: Can't connect in MySQL Workbench?
A: 1. Verify MySQL service is running
   2. Check your root password
   3. Use localhost not 127.0.0.1
   4. Port must be 3306

Q: Tables still don't show?
A: 1. Make sure you ran setup-database-workbench.sql
   2. Click refresh (üîÑ) in schemas panel
   3. Expand "pharmac_db" schema
   4. If still empty, run script again

Q: App still uses JSON storage?
A: 1. Check .env file has correct password
   2. Restart node server completely
   3. Check server logs for errors
